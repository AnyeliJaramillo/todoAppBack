# 🚀 TodoAppDev

Sistema web Full Stack para la gestión de proyectos y tareas desarrollado con Angular, Node.js, Express y MongoDB.

La plataforma permite administrar proyectos, usuarios y tareas mediante roles, prioridades visuales y reportes dinámicos.

---

## 🔧 Backend

## 🎨 Frontend

# ⚙️ Instalación del proyecto

## 🔧 Backend

Clonar repositorio:

```bash
git clone https://github.com/AnyeliJaramillo/todoAppBack.git
```

Entrar al proyecto:

```bash
cd todo-app-back
```

Instalar dependencias:

```bash
npm install
```

Ejecutar servidor:

```bash
npm start
```

Servidor local:

```txt
http://localhost:3000
```

---

## 🎨 Frontend

Clonar repositorio:

```bash
git clone https://github.com/AnyeliJaramillo/todoAppFront.git
```

Entrar al proyecto:

```bash
cd todo-app-front
```

Instalar dependencias:

```bash
npm install
```

Ejecutar aplicación:

```bash
ng serve
```

Aplicación local:

```txt
http://localhost:4200
```
